CareerBoost AI

IT Career Fit Prediction & Guidance System

Project Overview

CareerBoost AI is a Machine Learning based application that predicts a candidate's suitability for various IT job roles based on their skills, education, experience, projects, certifications, and communication abilities.

Features

- Job Fit Prediction
- Skill Gap Analysis
- Career Recommendations
- Alternative Career Suggestions
- Interactive Streamlit User Interface

Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit
- Joblib

Machine Learning Model

Random Forest Classifier

Dataset

Custom synthetic dataset containing multiple IT job roles, required skills, educational qualifications, experience levels, projects, certifications, and selection outcomes.
